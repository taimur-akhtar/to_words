# ToWords

This Gem converts numbers into words.

e.g.

1.to_words    # one
100.to_words  # one hundred
101.to_words  # one hundred and 1

## Installation

Add this line to your application's Gemfile:

    gem 'to_words'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install to_words
